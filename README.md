WOA application
=

Requirements (dev)
=
`go` installed on your local machine via homebrew or your favourite package manager

Usage
== 
`go run . <distance in Km>`
like
`go run . 2000`

Example output
```
              Kamenz |   551|
       Leipzig Halle |   657|
              Erfurt |   702|
             Hamburg |   933|
              Weelde |  1103|
            Alderney |  1576|
 Manchester Woodford |  1627|
```
